# sub_seek_for_android

【サブシーク for Androidについて】<br>
このシステムはデジタル庁「JGrants」APIを利用した検索システムです。 <br>
このシステムはどなたでも自由にご利用できます。 <br>
収集、掲載する情報の正確性には注意を払ってはおりますが、情報の正確性を保証するものではありません。<br>
<br>
※初回リリース版(v01.0.0)です

・補助金検索機能<br>
・補助金詳細表示機能<br>
・スマートフォン向けUI対応<br>
